# Gorion's Last Stand - Mod for Baldur's Gate Enhanced Edition

**Version:** 0.2  
**Author:** MephistoSatanDevil  
**Compatibility:** Baldur's Gate: Enhanced Edition Trilogy (EET)

## Introduction

This MOD aims to improve the death sequence of Gorion in the prologue of Baldur's Gate.

## Features

- Gorion will self-destruct when dying to prevent Sarevok from searching for the protagonist
- Adds unique dialogue and visual effects
- Maintains the original storyline flow unchanged

## Installation Instructions

1. Extract the MOD files to your game directory
2. Run `setup-GorionLastStand.exe`
3. Follow the prompts to complete the installation
